# rn-braintree

Upgraded wrapper of android and ios braintree sdk

## Installation

```sh
npm install rn-braintree
```

## Usage

```js
import RnBraintree from "rn-braintree";

// ...

const result = await RnBraintree.multiply(3, 7);
```
