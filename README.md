# rn-braintree

Upgraded wrapper of android and ios braintree sdk

## Installation

```sh
npm install rn-braintree-xplat
```

```sh
yarn add rn-braintree-xplat
```

## Usage

```js
import RnBraintree from 'rn-braintree-xplat'

// ...

const result = await RnBraintree.multiply(3, 7)
```
